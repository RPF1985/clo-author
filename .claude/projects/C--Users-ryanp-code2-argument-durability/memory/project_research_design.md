Project studies whether hybrid (normative + practical) political arguments are more fragile under partial disconfirmation than pure arguments. Three key hypotheses: (1) hybrid fragility when one leg is removed, (2) symmetry across normative/practical disconfirmation, (3) loss > gain asymmetry in argument structure.

**Why:** Novel theoretical contribution about argument structure durability — distinct from framing effects, motivated reasoning, or domain-specific war support literatures. User considers novelty likely but wants thorough confirmation via literature review.

**How to apply:** Paper scope is deliberately broad (not domain-specific). Experimental vignettes span war, trade, climate, immigration. Observational component is suggestive (elite rhetoric coding + opinion shifts), not a second causal claim. Literature novelty confirmation is highest priority for librarian.
